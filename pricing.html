What do I get?

More than you expect

  • Capacitive Erlang Applications

    All rebared and configured proven Erlang apps. We selected high reliable well known in Erlang community applications you can use immediately. Erlang provides best capacity per unit index from any other VM and GC platforms like JVM and CLR, that means your cost of ownership will be lower. Check out our Featured Erlang Applications to build your first Erlang release with Voxoz.

    Instance Images Versioning

    At any time you will be able to commit snapshots of your Erlang nodes, download to run locally or access via SSH. We provide full control to lightweight LXC instances. Because of using Linux containers you will get zero performance downgrade of host resources.

  • Flexible Control with Zero Complexity

    You could control your cloud with Voxoz dashboard using site, by using command line application or REST API. You will be able to add instances on the fly and monitor them with our Mandala Event Processor and HAL 9000 inspired SNMP monitor.

    Application Bones Included

    With our service subscription you will receive reference implementation of master application you want to build. You can rely on proven in production used by our clients and opensourced application bones. We have three major types of application you can use at start: Blog, App Store and Social App which are a very clean and production build on top of enterprise grade products like Riak and RabbitMQ, which are automatically included in your release subscription.
